# 可移除/禁用的服务与组件

> 生成日期：2026-06-18  
> 操作方式：`services.msc` 找到服务 → 右键属性 → 启动类型改为"手动"或"禁用"

---

## 一、建议直接禁用（无使用价值）

| 服务名 | 显示名 | 原因 |
|--------|--------|------|
| `Flash Helper Service` | Adobe Flash Helper | Flash 已于 2020 年 12 月彻底终止，此服务完全无用 |
| `MapsBroker` | Downloaded Maps Manager | Windows 自带地图离线服务，几乎无人使用 |
| `DiagTrack` | Connected User Experiences and Telemetry | Windows 遥测服务，持续向微软发送使用数据，隐私敏感 |
| `simptcp` | Simple TCP/IP Services | 上世纪遗留协议（echo/chargen），现代场景无需求 |
| `AiXiaobaoSvr` | Tencent AI小包服务 | 腾讯 AI 助手常驻后台（路径：E:\Program Files\Tencent\Androws），不主动使用可禁用 |
| `GooglePlayGamesServices-25.12.281.0` | Google Play Games 旧版 | 与 26.1 版本并存，旧版可直接禁用 |

## 五、一次性批量操作命令

以管理员身份运行 PowerShell，粘贴执行：

```powershell
# 禁用无用服务
$disable = @(
    'Flash Helper Service', 'MapsBroker', 'DiagTrack', 'simptcp',
    'AiXiaobaoSvr', 'GooglePlayGamesServices-25.12.281.0'
)
foreach ($s in $disable) { sc.exe config $s start= disabled }

# NI 服务改为手动
$niServices = @(
    'lkClassAds','lkTimeSync','mxssvr','NI SystemLink Message Bus',
    'NIApplicationWebServer','niauth','NIDomainService','niLXIDiscovery',
    'NINetworkDiscovery','nipxicmsvc','niroco','NiSvcLoc',
    'NISystemWebServer','NITaggerService','UALDS'
)
foreach ($s in $niServices) { sc.exe config $s start= demand }

# Siemens 服务改为手动
$siemensServices = @(
    'S7TraceServiceX','SiemensTiaAdminV3','TraceConceptX','umscsvc',
    'almservice','SIMATIC PnDiscovery Service'
)
foreach ($s in $siemensServices) { sc.exe config $s start= demand }

# SolidWorks 服务改为手动
$swServices = @(
    'SolidWorks Flexnet Server','SolidWorks Licensing Service',
    'SWVisualize2026.Queue.Server'
)
foreach ($s in $swServices) { sc.exe config $s start= demand }
```

## 六、MacType 与文字大小 120% 失效修复

**问题**：每次重启后"设置 → 辅助功能 → 文字大小"的 120% 设置不生效，需手动重新应用。

**根因**：MacType 服务（`AUTO_START`）在登录早期启动并接管 GDI 渲染，其 `RedrawDelay=5000` 导致初始化需 5 秒以上；Windows 的文字大小广播消息（`WM_SETTINGCHANGE`）恰在此窗口内触发，被 MacType 钩子干扰，第一次广播失效。

**修复步骤**：

**第一步**：脚本已创建于 `C:\Users\ybs15\.scripts\reapply-textscale.ps1`

**第二步**：以管理员身份运行 PowerShell，执行以下命令注册登录延迟任务：

```powershell
Register-ScheduledTask -TaskName "ReapplyTextScale-MacType" -Action (New-ScheduledTaskAction -Execute "powershell.exe" -Argument "-WindowStyle Hidden -NonInteractive -ExecutionPolicy Bypass -File `"C:\Users\ybs15\.scripts\reapply-textscale.ps1`"") -Trigger (New-ScheduledTaskTrigger -AtLogOn -User "ybs15") -Settings (New-ScheduledTaskSettingsSet -ExecutionTimeLimit (New-TimeSpan -Minutes 2) -MultipleInstances IgnoreNew) -Description "MacType 初始化后重新应用文字大小 120%" -Force
```

**效果**：登录后约 8 秒自动重新广播设置，无需手动操作。
