# 开机后只使用内屏，命令和开合都可以唤醒，内屏亮
# 开机后只使用外屏，命令可以唤醒，合盖并不会进入休眠
# 开机后只使用扩展模式(内屏+外屏)，命令可以唤醒，合盖不会进入休眠，而是内屏关闭，转为仅外屏模式，开盖后回到扩展模式，内屏也亮
# 开机后先使用扩展模式(内屏+外屏)，命令可以唤醒，合盖情况同上，此时再换成只使用内屏(插着外接视频线)，命令可以正常唤醒，合盖后没有进入休眠，而是转为了仅外屏模式，此时再开盖自动转为扩展模式！
# 开机后先使用扩展模式(内屏+外屏)，命令可以唤醒，合盖情况同上，此时再换成只使用内屏(然后移除外接视频线，模拟将要出门的情景)，命令和开合盖都可以正常唤醒，不过开盖后需要敲击一下键盘才能唤醒
